How to play.

  • start server

Terminal 1
| % ruby name.rb druby://yourhost:7640
| druby://yourhost:7640
| [return] to exit

  • start client

Terminal 2
| % ruby namec.rb druby://yourhost:7640
| #<DRb::DRbObject:0x40164174 @uri="druby://yourhost:7640", @ref="seq">
| #<DRb::DRbObject:0x40163c9c @uri="druby://yourhost:7640", @ref="mutex">
| 1
| 2
| [return] to continue

  • restart server

Terminal 1
type [return]
| % ruby name.rb druby://yourhost:7640
| druby://yourhost:7640
| [return] to exit

  • continue client

Terminal 2
type [return]
| 1
| 2
