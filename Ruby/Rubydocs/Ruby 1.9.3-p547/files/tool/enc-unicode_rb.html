Creates the data structures needed by Onigurma to map Unicode codepoints to property names and POSIX character classes

To use this, get UnicodeData.txt, Scripts.txt, PropList.txt, PropertyAliases.txt, PropertyValueAliases.txt, DerivedCoreProperties.txt, and DerivedAge.txt from unicode.org. (unicode.org/Public/UNIDATA/) And run following command. ruby1.9 tool/enc-unicode.rb data_dir > enc/unicode/name2ctype.kwd You can get source file for gperf. After this, simply make ruby.
